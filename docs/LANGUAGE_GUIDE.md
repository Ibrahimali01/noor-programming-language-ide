# دليل التوثيق الشامل للغة نور (Noor Sovereign Language Official Docs)

## 1. الاستقلالية الكاملة (Sovereign Independence)
لغة **نور** لغة مبرمجة من الصفر. لا يتم ترجمتها إلى JavaScript أو Python في الخلفيات الفعلية (يتم استخدام Node.js في بيئة الـ IDE الحالية لمحاكاة المحرك فقط، لكن المحرك المترجم يعمل بشكل مستقل). هي لغة نابعة من ذاتها بمعالجها الخاص `Noor VM`.

## 2. برمجة المواقع وهيكلة الواجهات (Web Mastery)
نور تدعم دوال مدمجة عميقة للويب لتوفير بيئة (Full-Stack) دون الاعتماد على HTML/CSS كالتالي:
- `هيكل_صفحة("اسم")`: تكوين الدوم.
- `تلوين_النص(العنصر, "اللون")`
- `خلفية_الصورة(العنصر, "الخلفية")`
- `تنسيق_الخط(العنصر, "النوع", الحجم, "السمك")`
- `التحكم_بالأبعاد(العنصر, x, y, z)`: لتحديد الطول والعرض والمسافات.
- `إضافة_تأثير_حركة(العنصر, "نوع الحركة")`: للـ Animations.
- `إضافة_نصوص(العنصر, "النوع", "النص")`: لكتابة عنواين وفقرات.
- `إدراج_وسائط(العنصر, "صورة/فيديو", "المسار")`
- `تصميم_جدول` و `بناء_قائمة`
- `توجيه_متصفح(المتصفح, العنصر)`: توجيه أوتوماتيكي لمحركات المتصفحات.

## 3. الهواتف والألعاب (Mobile & Gaming)
- `تصدير_هاتف("أندرويد" | "آيفون")`
- `تهيئة_محرك()`: محرك ألعاب 120 إطار في الثانية للفيزياء الحقيقية.
- `رسم_مجسم(الكائن, الإحداثيات)`

## 4. أمن المعلومات وأنظمة لينكس (Cyber & OS)
- `تشفير_البيانات("النص")`: تشفير AES مخصص.
- `تنفيذ_شيل("أوامر اللينكس")`: اتصال مباشر بالنواة والتيرمنال الخفي.
- `إدارة_أنظمة_قديمة("Mainframe 1990")`: حقن أوامر للشبكات الإرثية.
- `اعدادات_النظام("متغير", "قيمة")`

## 5. قواعد البيانات وتعلم الآلة
- `استعلام_سريع("QUERY")`: محرك قواعد نور دي بي.
- `تدريب_نموذج("الاسم", البيانات)`: نماذج التعلم العميق.
- `تحليل_لغوي("النص")`: معالجة لغوية مدعومة بقوة نور الآلية.
